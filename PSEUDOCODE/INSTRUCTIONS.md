# US National Park Directory

## User Story

```
AS A user
I WANT to view information on US National Parks
SO THAT I can learn about the services each park offers, and any recent news pertaining to the park
```

## Acceptance Criteria

```
GIVEN a park dashboard with a search input
WHEN I search for a park
THEN I am presented with the park name, description, a wikipedia link, and articles
WHEN I click the Wikipedia link
THEN I am redirected to the wikipedia page for the related search
WHEN I view the articles section
THEN I am presented with NY Times articles for the park
WHEN I click clear
THEN I am all content is removed, and I can make a fresh search
```
